Java Class 공부

클래스(Class)

클래스(Class)는 일종의 틀입니다. 쿠키같은 과자를 만들 때, 틀모양에 따라 쿠키를 찍어낼 수 있다. 그런데, 재료에 따라 같은 모양(틀)이어도 서로 다른 쿠키가 완성된다.

그리고 과자틀에 의해서 만들어진 과자들을 객체(Object)라고 한다.

Animal Class가 있고, main에서 Animal cat = new Animal(); 코드에 의해 cat이라는 객체가 생성된다.

메서드(Method)

클래스에는 객체 변수와 더불어 메서드(Method)라는 것이 있다. 메서드는 클래스 내에 구현된 함수를 의미하는데 보통 함수라고 말하지 않고 메서드라고 말한다.

위의 코드에서는 Animal Class 안에 name이라는 객체변수가 있는데, 어디에도 할당이 없기 때문에 name을 print해보면, null이 나오게 된다.

이제 메서드를 이용하여 Animal 클래스의 객체 변수인 name 에 값을 대입해보자.

생성자(Constructor)

메서드명이 클래스명과 동일하고 리턴 자료형을 정의하지 않는 메서드를 생성자(Constructor)라고 한다.

생성자의 규칙

  1. 클래스명과 메서드명이 동일하다.
  2. 리턴타입을 정의하지 않는다. (void도 사용하지 않는다.)

생성자는 객체가 생성될 때 호출된다. 즉, 생성자는 다음과 같이 new 키워드가 사용될 때 호출된다.
또한, 생성자는 메서드와 마찬가지로 다양한 입력을 받을 수 있다.
